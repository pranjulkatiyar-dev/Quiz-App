# Quiz-App

An interactive quiz application built using modern JavaScript, HTML5, and CSS3.
This project demonstrates DOM manipulation, event handling, and dynamic UI updates.

# Features
📋 10+ multiple-choice questions
🔄 Smooth question navigation
✅ Submit-based quiz system
📊 Final score displayed after submission
🎯 Clean and intuitive user interface

# What I Learned
* Structured complete quiz flow including:
    Question rendering
    Answer selection logic
* Implemented JavaScript DOM manipulation for dynamic updates
* Configured event handling for user interactions
* Managed navigation between questions efficiently
* Optimized UI using CSS3 for better user experience

# Tech Stack
JavaScript (ES6+)
HTML5
CSS3
